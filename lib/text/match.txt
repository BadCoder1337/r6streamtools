**Помощь по команде `match`:**
1.`$$match create <team1Code> <team2Code> <matchType> <banOps>` - создать матч
	__teamCode__ - коды команд
	__matchType__ - режим матча: bo1, bo3 или bo5
	__banOps__ - включить режим бана оперативников: yes или no
2.`$$match info <id>` - параметры матча
	__id__ - ID матча
3.`$$match edit <id> <prop> <value>` - редактировать матч
	__id__ - ID матча
	__prop__ - редактируемое значение(`info` для того чтобы узнать значения)
	__value__ - новое значение
4.`$$match start <id>` - начать матч(голосование за карты/оперативников)
	__id__ - ID матча
